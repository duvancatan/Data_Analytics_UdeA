# **Analítica de Datos - Universidad de Antioquia**

### **Descripción**

Este repositorio contiene el material del curso de Analítica de Datos, enfocado en el desarrollo de habilidades prácticas para la manipulación, análisis e interpretación de datos en contextos reales. A lo largo del curso, se construye una base sólida que conecta herramientas fundamentales con metodologías modernas utilizadas en la industria y la academia.


### **Contenidos**

⚙️ Terminal y Shell

🔄 Git y GitHub

📊 EDA y Visualización

🤖 Modelos Supervisados

🔍 Modelos No Supervisados

🚀 Puesta en Producción


### **Tecnologías**

Python · Pandas · NumPy · Matplotlib · Seaborn · Scikit-learn · Git


### **Estructura**
data/ | notebooks/ | src/ | models/ | reports/


### **Proyecto Final**

Aplicación completa de analítica: desde la exploración hasta la implementación de un modelo funcional.
